# Bundled fonts

## Geist (`Geist-Variable.woff2`)

- **Designer:** Vercel
- **License:** SIL Open Font License 1.1 (OFL-1.1) — free to bundle and redistribute with the app.
- **Source:** https://github.com/vercel/geist-font (also `npm: geist`).

DotFlow bundles the variable font locally so the UI renders identically offline (no CDN / network at
runtime). Used as the app's primary interface typeface (see `src/App.css` `@font-face`).
